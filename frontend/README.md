# Frontend — Leeshan's app

This folder is Leeshan's Next.js app. **Build it from `ORKESTR_FRONTEND.md`**
(14 screens, cream/coral theme, her `lib/api.js`).

The old dark single-page skeleton has been retired — the team chose the full
14-screen product with Leeshan's warm theme.

The backend (repo root) already speaks the exact API contract in her guide §4/§6.
Point `NEXT_PUBLIC_API_URL=http://localhost:8000` and flip `NEXT_PUBLIC_USE_MOCK=false`
when the backend is running. Seeded session id for the demo: **ORK-001**.
